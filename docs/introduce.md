#Introduce yefm file manager

###Features:

- Simple GUI
- Multi-tabs
- 2 panes
- Expanding the window height on second pane visible
- Colored status text
- With mime-open settings dialog
- Dnd folder or bookmark to add new tab
- Drag a tab to another pane
